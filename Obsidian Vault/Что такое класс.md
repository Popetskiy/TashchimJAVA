
Класс в Java — это шаблон или чертеж для создания объектов. Он описывает состояние объекта через поля (переменные, например, name, age) и его поведение через методы (функции, например, sayMeow()). Для инициализации объектов при их создании используются специальные методы — конструкторы.
**Класс** — это, по сути, шаблон для объекта. Он определяет, как объект будет выглядеть и какими функциями обладать. Каждый объект является объектом какого-то класса.

```
public class Cat {
 String name; int age;
}
```

Создадим классический пример с классом `Cat` (Кот). На его примере разберем, как объект получает свойства и как он связан с тем самым базовым классом `Object`.

## Пример на языке Java

```java
public class Cat {
    // 1. Поля (свойства) объекта
    String name;
    int age;

    // 2. Конструктор (создает кота с заданными параметрами)
    public Cat(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // 3. Методы (поведение)
    public void meow() {
        System.out.println(name + " говорит: Мяу!");
    }

    // 4. Переопределение метода из базового класса Object
    @Override
    public String toString() {
        return "Кот [Имя: " + name + ", Возраст: " + age + "]";
    }
}
```

## Как это работает в коде:

```java
public class Main {
    public static void main(String[] args) {
        // Создаем экземпляр (объект) класса Cat
        Cat myCat = new Cat("Барсик", 3);

        // Используем собственный метод
        myCat.meow(); 

        // Вызываем toString() (который мы переопределили из класса Object)
        System.out.println(myCat.toString()); 
    }
}
```

## Что здесь важно для понимания:

1. Класс — это чертеж. Сам по себе `Cat` — это просто описание того, какими бывают коты.
2. Объект (`myCat`) — это экземпляр. Это конкретный «Барсик» с конкретным возрастом, живущий в памяти компьютера.
3. Наследование от `Object`. Заметьте, мы не писали `extends Object`, но метод `toString()` нам доступен и мы его переопределили (изменили стандартное поведение), чтобы вместо непонятного адреса в памяти выводилось красивое описание кота.
