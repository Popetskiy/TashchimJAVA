Представим, что класс — это чертеж автомобиля, а объекты — это сами машины, собранные по этому чертежу.
[[Что такое объект]]

- Обычные методы — это функции конкретной машины. Например, «нажать на тормоз» или «открыть багажник». Ты не можешь нажать на тормоз «чертежа», тебе нужна реальная машина.
## - Статические методы 
— это функции самого «чертежа» или всего автозавода. Например, метод «узнать общее количество выпущенных машин» или «перевести мили в километры». Тебе не нужно садиться в конкретный автомобиль, чтобы это сделать.

## Главное в трех пунктах:

1. Принадлежат классу, а не объекту. Ты пишешь `Math.sqrt(16)`, а не создаешь сначала «математику» через `new Math()`.
2. Видят только «общее». Статический метод не знает, какого цвета сиденья у конкретной машины, он знает только то, что общее для всех (статические переменные).
3. Всегда доступны. Они существуют сразу, как только программа запустилась, даже если ты не создал ни одного объекта этого класса.

Самый частый пример:  
Утилиты. Когда тебе нужно просто посчитать что-то или проверить строку, ты используешь статический метод, чтобы не плодить лишние объекты в памяти.

Пример:
```
class Car {
    static int carCount = 0; // Общая переменная для всех машин
    String color;           // Цвет конкретной машины

    Car(String color) {
        this.color = color;
        carCount++; // Увеличиваем общий счетчик при создании новой машины
    }

    // СТАТИЧЕСКИЙ метод: относится ко всем машинам сразу
    public static void showTotalCars() {
        System.out.println("Всего машин выпущено: " + carCount);
        // System.out.println(color); // ОШИБКА! Статический метод не знает, про какой цвет речь
    }

    // ОБЫЧНЫЙ метод: относится к конкретной машине
    public void drive() {
        System.out.println("Эта " + color + " машина поехала.");
    }
}

public class Main {
    public static void main(String[] args) {
        // Вызываем статический метод напрямую через КЛАСС (объект не нужен)
        Car.showTotalCars(); 

        // Создаем объекты (конкретные машины)
        Car myCar = new Car("Красная");
        Car yourCar = new Car("Синяя");

        // Вызываем обычные методы через ОБЪЕКТЫ
        myCar.drive();
        yourCar.drive();

        // Снова вызываем статический метод — данные обновились для всех
        Car.showTotalCars();
    }
}

```
