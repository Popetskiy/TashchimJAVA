## Что это?
В Java **сигнатура** — это ==«паспорт» или «уникальный отпечаток» метода==. По ней компьютер понимает, какой именно метод ты вызываешь, если их в программе много.

<font color="#f79646">**Сигнатура = Имя + Типы того, что в скобках.**</font>

Сигнатура состоит всего из **двух вещей**:

1. **Имя метода.**
2. **Список параметров** (их типы и порядок).

Что НЕ входит в сигнатуру:

Это важно запомнить: **тип возвращаемого значения** (например, `int` или `void`) и **модификаторы** (`public`, `static`) в сигнатуру **не входят**.

---

## Примеры для наглядности:

Представим метод:  
`public static int sum(int a, int b) { ... }`

Его сигнатура: **`sum(int, int)`**

| Метод                         | Сигнатура                | Пояснение                                           |
| ----------------------------- | ------------------------ | --------------------------------------------------- |
| `void print(String s)`        | **`print(String)`**      | Имя и один текст                                    |
| `void print(int n)`           | **`print(int)`**         | Имя то же, но тип другой — это **другая** сигнатура |
| `void print(int n, String s)` | **`print(int, String)`** | Порядок и количество важны                          |
