Эти четыре метода — база для работы со строками в Java. Поскольку `String` — это объект, мы вызываем их через точку после имени переменной.

## 1. `length()` — Длина строки

Возвращает количество символов в строке (тип `int`).

- Важно: Считает всё — пробелы, знаки препинания и спецсимволы.
- Пример:
    
    ```java
    String s = "Кот";
    System.out.println(s.length()); // Выведет: 3
    ```
    

## 2. `charAt(int index)` — Символ по индексу

Позволяет достать конкретную «букву» из строки.

- Нюанс: Отсчет индексов всегда начинается с 0.
- Ошибка: Если запросить индекс больше, чем длина строки, программа «упадет» с ошибкой `IndexOutOfBoundsException`.
- Пример:

    ```java
    String s = "Java";
    char letter = s.charAt(0); // Достаем первую букву
    System.out.println(letter); // Выведет: J
    ```


## 3. `concat(String str)` — Склеивание (Конкатенация)

Присоединяет одну строку к концу другой.

- Особенность: Метод не меняет старую строку (она неизменяема!), а создает новую.
- Альтернатива: Чаще используют обычный плюс `+` (`"А" + "Б"`), но `concat` — это «официальный» метод.
- Пример:

    ```java
    String s1 = "Привет, ";
    String s2 = "мир!";
    String result = s1.concat(s2); // "Привет, мир!"
    ```


## 4. `equals(Object anObject)` — Сравнение по содержанию

Самый важный метод для тестировщика и разработчика.

- Почему не `==`? Оператор `==` сравнивает адреса в памяти (ссылки). Метод `equals` сравнивает именно текст внутри объектов.
- Пример:

    ```java
    String s1 = new String("Тест");
    String s2 = new String("Тест");
    
    System.out.println(s1 == s2);      // false (разные объекты в памяти)
    System.out.println(s1.equals(s2)); // true (текст одинаковый)
    ```
