Статическая переменная — это ==переменная, которая принадлежит всему классу сразу, а не какому-то конкретному объекту==.

Если обычные переменные у каждого объекта свои (у одного кота имя «Барсик», у другого — «Мурзик»), то статическая переменная — одна на всех.

## Главные фишки:

1. Общее хранилище: Если один объект изменит статическую переменную, она изменится для всех остальных объектов этого класса.
2. Вызов через класс: К ней обращаются по имени класса: `Human.totalCount`, а не через `myObject.totalCount`.
3. Живет всегда: Она появляется в памяти в момент старта программы (при загрузке класса) и исчезает только при её завершении.

## Пример, чтобы стало совсем понятно:

Представим класс `Soldier` (Солдат).

- Обычная переменная: `name`. У каждого солдата своё имя.
- Статическая переменная: `commanderName`. Командир у всей роты один. Если командир сменится, он сменится сразу для всех солдат.

```java
class Soldier {
    String name;               // Обычная
    static String commander;   // Статическая
}

public class Main {
    public static void main(String[] args) {
        Soldier.commander = "Полковник Иванов"; // Установили один раз для всех

        Soldier s1 = new Soldier();
        s1.name = "Иван";

        Soldier s2 = new Soldier();
        s2.name = "Петр";

        // Оба солдата знают, что их командир — Иванов
        System.out.println(s1.commander); // Иванов
        System.out.println(s2.commander); // Иванов
    }
}
```

## Зачем их используют?

- Счетчики: Например, считать, сколько всего объектов этого класса было создано.
- Константы: Те самые `static final`
- Настройки: Глобальные параметры, которые должны быть одинаковыми для всей программы.
