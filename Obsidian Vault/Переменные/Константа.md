Константа — это ==переменная, значение которой нельзя изменить после того, как его один раз записали==.
[[Виды констант]]

В Java для создания константы используются два ключевых слова: `static` и `final`.

Константы обычно объявляются с помощью final.
Если константа общая для всего класса, используют static final.

Пример:
static final int LIMIT_SUM = 100;
Правила именования констант:
все буквы в верхнем регистре;
cлова разделяются через  _ .

Примеры:
LIMIT_SUM
MAX_USERS
DEFAULT_TIMEOUT

## Как это выглядит в коде:

```java
public class Settings {
    public static final int MAX_USERS = 100;
}
```

## Разбор по словам:

1. `final` — это «замок». Он говорит: «Это значение — последнее, его нельзя перезаписать». Если ты попробуешь написать `MAX_USERS = 200`, программа не скомпилируется.
2. `static` — делает константу «общей». Нам не нужно создавать объект класса `Settings`, чтобы узнать лимит пользователей. Мы просто пишем `Settings.MAX_USERS`.

---

## Главные правила констант:

- Имя капсом: По правилам Java (Code Style) константы всегда пишутся большими буквами, а слова разделяются подчеркиванием: `USER_TIMEOUT_SECONDS`.
- Сразу даем значение: Обычно значение присваивается сразу при объявлении.
- Экономия памяти: Благодаря `static` константа существует в памяти в одном единственном экземпляре.

## Зачем они нужны?

Чтобы не использовать в коде «магические числа».

- Плохо: `if (age < 18)` — через месяц ты забудешь, что значит 18.
- Хорошо: `if (age < MIN_ADULT_AGE)` — сразу понятно, о чем речь. Если возраст совершеннолетия изменится, можно поправить его в одном месте, а не по всему коду.
