В Java ссылочные типы (Reference Types) — ==это всё, что не является примитивом==. Если примитив — это сама «коробка с числом», то ссылочный тип — это пульт дистанционного управления от этой коробки.

**<font color="#9bbb59">Главные отличия и особенности:</font>**

## 1. Что хранится в переменной?

В переменной ссылочного типа хранится не само значение (данные), а адрес в памяти (ссылка), по которому эти данные лежат в «Куче» (Heap). [[Куча (Heap)|это]]

- Примитив: `int a = 5;` — в ячейке `a` лежит число `5`.
- Ссылка: `String s = "Hello";` — в ячейке `s` лежит адрес (например, `0x778`), а по этому адресу в памяти уже лежат буквы «H-e-l-l-o».

## 2. Какие бывают ссылочные типы?

Их бесконечно много, но основных групп четыре:

1. Классы (например, `String`, `Scanner`, `Integer` или ваш собственный `User`).
2. Интерфейсы (например, `List`, `Runnable`).
3. Массивы (даже массив примитивов `int[]` — это уже ссылочный тип!).
4. Перечисления (`enum`).

## 3. Значение по умолчанию: `null`

Если примитив всегда равен какому-то числу (0, false), то ссылочная переменная может быть равна `null`.

- `null` означает «адреса нет», «пульт ни к чему не привязан».
- Если  попытаться вызвать метод у такой переменной (например, `s.length()`, когда `s = null`), программа упадет с ошибкой NullPointerException.

## 4. Копирование ссылок

Это самый важный момент для понимания «управления приложением»:

- Когда  пишешь `int b = a;`, создается копия числа.
- Когда  пишешь `User user2 = user1;`, не создается новый пользователь. Создается второй пульт от того же самого объекта. Если через `user2` изменить имя, оно изменится и в `user1`, потому что объект один и тот же.

## 5. Размер в памяти

В отличие от примитивов, размер которые расписывали (1, 2, 4, 8 байт), сама ссылка (адрес) в Java обычно занимает:

- 4 байта на 32-битных системах (или на 64-битных с включенным сжатием ссылок CompressedOops, что в Java по умолчанию).
- 8 байт на чистых 64-битных системах.
- _Важно:_ Это размер только «адреса». Сам объект в Куче может весить хоть несколько гигабайт.

Сравнение в таблице:

|Характеристика|Примитивы (`int`, `char`...)|Ссылочные типы (`String`, `Object`...)|
|---|---|---|
|Где лежат данные|Прямо в переменной (в Стеке)|В объекте (в Куче)|
|Что в переменной|Значение|Адрес в памяти|
|Пустое значение|Невозможно|`null`|
|Методы|Нет методов|Можно вызывать методы через `.`|
