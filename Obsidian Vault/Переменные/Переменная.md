Переменная — это именованная область памяти, в которой хранится значение.
Через переменные программа хранит и изменяет данные во время работы.

Примитивные типы имеют:
- фиксированный формат данных,
- ограниченный размер в памяти,
- свой диапазон допустимых значений.

Виды переменных в Java
1. Переменные экземпляра объекта

Объявляются внутри класса, но вне методов, конструкторов и блоков.
Создаются вместе с объектом и существуют, пока существует объект.

Особенности: у каждого объекта свои значения таких переменных;
доступны внутри методов и конструктора этого объекта.

Пример:
```
class User {
    String name;
    int age;
}
```

2. **Локальные переменные**

Объявляются внутри метода, конструктора или блока кода.
Существуют только во время выполнения этого блока.

**Особенности:**
видны только внутри своего блока;
вне метода или блока недоступны;
после завершения блока перестают существовать.

Пример:
```
void printAge() {
    int age = 25;
    System.out.println(age);
}
```

3. Статические поля класса

Это переменные, объявленные с static.
Они принадлежат всему классу, а не отдельному объекту.

**Особенности:**
одна общая переменная для всех объектов класса;
можно обращаться без создания объекта.

Пример:
```
class User {
    static int count = 0;
}
```
