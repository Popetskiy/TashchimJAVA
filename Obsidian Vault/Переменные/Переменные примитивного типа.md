В Java **примитивы** — это ==самые простые, «атомарные» типы данных, которые встроены в сам язык==.

Если объект (например, `String` или `Scanner`) — это сложная конструкция, то примитив — это **чистое значение**.

Вот 3 главных отличия, которые объясняют, что это такое:

1. **Хранят значение напрямую:**  
    В отличие от объектов, которые хранят «адрес» (ссылку) на данные где-то в памяти, примитивная переменная содержит **само число или символ** прямо в той ячейке памяти, которая за ней закреплена.
2. **Не являются объектами:**  
    У примитивов нет методов. Вы не можете написать `int x = 5; x.toString();` — это вызовет ошибку. Они существуют для максимальной скорости и экономии памяти.
3. **Всегда имеют значение по умолчанию:**  
    Если вы создадите поле в классе и не укажете значение, Java сама запишет туда «ноль» (для чисел это `0`, для `boolean` — `false`). Примитив не может быть равен `null`.

**Всего в Java 8 примитивов:**

- **Целые:** `byte`, `short`, `int`, `long`
- **Дробные:** `float`, `double`
- **Символ:** `char`
- **Логика:** `boolean`

**Метафора:**  
Представим, что **объект** — это огромная посылка с инструкцией, весом и курьером, а **примитив** — это просто число, написанное мелом на доске. Это быстро, дешево и занимает минимум места.

### Формат данных, размер, диапазон значений

1. Целочисленные типы (Целые числа)

Используют кодирование «дополнительный код» (signed), поэтому диапазон всегда смещен в отрицательную сторону на единицу больше.

|Тип|Размер|Точный диапазон (от ... до ...)|
|---|---|---|
|**`byte`**|8 бит (1 байт)|от **-128** до **127**|
|**`short`**|16 бит (2 байта)|от **-32 768** до **32 767**|
|**`int`**|32 бита (4 байта)|от **-2 147 483 648** до **2 147 483 647**|
|**`long`**|64 бита (8 байт)|от **-9 223 372 036 854 775 808** до **9 223 372 036 854 775 807**|

2. Типы с плавающей точкой (Дробные числа)

Соответствуют стандарту IEEE 754.

|Тип|Размер|Точность|Диапазон (крайние значения)|
|---|---|---|---|
|**`float`**|32 бита (4 байта)|~7 знаков|от **1.4e-45** до **3.4028235e+38**|
|**`double`**|64 бита (8 байт)|~15-17 знаков|от **4.9e-324** до **1.7976931348623157e+308**|

3. Символьный тип

В Java используется кодировка **UTF-16**, поэтому `char` всегда занимает 2 байта.

|Тип|Размер|Диапазон / Значение|
|---|---|---|
|**`char`**|16 бит (2 байта)|от **0** до **65 535** (в кодах Unicode от `'\u0000'` до `'\uffff'`)|

4. Логический тип

Самый специфичный тип в Java.

|Тип|Размер|Значения|
|---|---|---|
|**`boolean`**|*Не определен строго|Только **true** или **false**|
