Переменная предоставляется нам как место хранения наших данных (или адрес места) 
У переменных примитивного типа есть определенный формат данных, размер, диапазон значений, которые могут храниться в памяти. 
Константа — это заданные, фиксированные значения, которые не должны меняться

### Место хранения и адреса

Например  компьютерная память — это **огромный склад** с миллионами пронумерованных ячеек.

Вот так  это работает на пальцах:

1. **Адрес памяти:** Каждая ячейка на этом складе имеет свой длинный и сложный номер (например, `0x7ffcd4`). Роботу-процессору удобно работать с такими цифрами, а человеку — нет.
2. **Переменная:** Это просто **понятный ярлык (имя)**, который ты вешаешь на эту ячейку. Вместо того чтобы запоминать «положи число 5 в ячейку №00451», говорим: «пусть в переменной `score` лежит 5».
3. **Место хранения:** Когда  создается переменная, компьютер выделяет  «коробку» нужного размера. Можешь класть туда данные, вынимать их или заменять на новые.

**Итог:** Переменная — это удобный псевдоним для конкретного места в памяти, чтобы можно было обращаться к своим данным по имени, а не по «прописке».
## Виды переменных

- переменные объекта - Переменные создаются лишь после того, как будет создан объект (то есть после создания объекта с помощью `new`). В объекте внутренние переменные всегда видимы для всех методов, конструктора или чего-либо внутри этого же объекта.

- локальные переменные - Данные переменные объявляются в методах, конструкторах или блоках . Они видимы лишь в пределах объявленного метода, конструктора или блока. То есть использовать переменную, например, в другом методе нельзя.

- переменные класса -данный вид переменных ещё называют статическими. Они объявляются со словом-модификатором `static`, но за пределами метода, конструктора или блока.
- 
Существует особый вид переменных — константы, которые могут быть глобальными, переменными. Константа — это заданные, фиксированные значения, которые не должны меняться. Их область видимости — вся программа.

## Переменные и константы

Переменная — это именованная область памяти, в которой хранится значение.
Через переменные программа хранит и изменяет данные во время работы.

Примитивные типы имеют:
- фиксированный формат данных,
- ограниченный размер в памяти,
- свой диапазон допустимых значений.

**Константа** — это значение, которое после задания не должно изменяться.

Виды переменных в Java
1. Переменные экземпляра объекта

Объявляются внутри класса, но вне методов, конструкторов и блоков.
Создаются вместе с объектом и существуют, пока существует объект.

Особенности: у каждого объекта свои значения таких переменных;
доступны внутри методов и конструктора этого объекта.

Пример:
```
class User {
    String name;
    int age;
}
```

2. **Локальные переменные**

Объявляются внутри метода, конструктора или блока кода.
Существуют только во время выполнения этого блока.

**Особенности:**
видны только внутри своего блока;
вне метода или блока недоступны;
после завершения блока перестают существовать.

Пример:
```
void printAge() {
    int age = 25;
    System.out.println(age);
}
```

3. Статические поля класса

Это переменные, объявленные с static.
Они принадлежат всему классу, а не отдельному объекту.

**Особенности:**
одна общая переменная для всех объектов класса;
можно обращаться без создания объекта.

Пример:
```
class User {
    static int count = 0;
}
```
### Константы в Java

Константы обычно объявляются с помощью final.
Если константа общая для всего класса, используют static final.

Пример:

static final int LIMIT_SUM = 100;
Правила именования констант:
все буквы в верхнем регистре;
cлова разделяются через  _ .

Примеры:
LIMIT_SUM
MAX_USERS
DEFAULT_TIMEOUT

**Кратко**
переменная экземпляра — хранит данные конкретного объекта;
локальная переменная — существует только внутри метода или блока;
переменная класса (static) — общая для всех объектов;
константа (final, часто static final) — неизменяемое значение.
